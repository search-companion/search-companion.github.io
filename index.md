## Welcome to search-companion

search-companion.org is building open source tools to help technical people to interact their systems with the open source search engine Solr ([https://solr.apache.org/](https://solr.apache.org/)). 
The functionality is based on apache camel ([https://camel.apache.org/](https://camel.apache.org/)) routes and enables easy deployment in apache karaf ([https://karaf.apache.org/](https://karaf.apache.org/)) as "search companion" karaf features.

### search-companion features

- data-import: alternative and robust solution to replace the deprecated solr dataimporthandler that imports data from a datasource into solr
